# How does the internet work?
## 인터넷은 어떻게 동작하는가
- 한줄로 굳이 표현하자면 클라이언트 - ISP - DNS - Server
- 2개의 컴퓨터가 통신이 필요하면 우리는 케이블을 연결
  - 그런데 N개의 컴퓨터를 서로 연결하려면?
  - 케이블이 N*N개만큼 필요하지
  - 그래서 Router가 필요한 것
- 우리끼리말고 다른 외부 네트워크에도 연결하자
  - 그래서 ISP가 필요
- IP만으로는 다른 컴퓨터를 찾기가 힘들잖아!
  - 그래서 DNS가 필요
